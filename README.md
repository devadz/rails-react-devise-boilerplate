# rails-react-devise-boilerplate

This is a basic boilerplate for a Rails project with user registration.

## Ingredients

* Ruby on Rails (4.2.4)
* PostgreSQL database
* RSpec testing framework
* Twitter bootstrap SCSS framework (v3.3.7)
* Font awesome (4.7.0)
* Devise gem for user registration
